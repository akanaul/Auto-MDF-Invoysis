# Documentação do Auto MDF InvoISys

Bem-vindo à documentação do **Auto MDF InvoISys**! Este software ajuda a automatizar processos relacionados ao MDF-e de forma simples e segura.

## O que é o Auto MDF InvoISys?

O Auto MDF InvoISys é uma ferramenta que automatiza tarefas repetitivas no sistema Invoisys, como preenchimento de formulários e geração de MDF-e. Ele usa uma interface gráfica fácil de usar, com botões e menus simples, para que você possa executar scripts sem precisar saber programar.

**Destaque:** A instalação é totalmente automática - o launcher `AutoMDF-Start.py` baixa e instala tudo que precisa quando você executa pela primeira vez. Para usar, basta dar duplo clique no launcher!

### Principais Recursos

- **Interface Gráfica Amigável:** Janela principal com botões para iniciar, parar e configurar.
- **Execução Segura:** O software verifica se tudo está correto antes de começar.
- **Logs Detalhados:** Registra o que acontece durante a execução para ajudar em caso de dúvidas.
- **Configurações Personalizáveis:** Ajuste tempos e opções sem mexer em códigos.
